Embedding with <binding-sim>

Include the script once, then drop the <binding-sim> tag anywhere. Each element is an independent, isolated instance (it wraps an iframe), so you can place several on one page without conflicts.

<script src="https://your-host/bindingsims/binding-sim.js"></script>

<binding-sim model="ligand"></binding-sim>
<binding-sim model="receptors" ext height="1000"></binding-sim>

Attributes: model (required — ligand, homodimer, ligands, receptors), ext (boolean, power-user features), theme (dark/light, same-origin), height / width (CSS length; a bare number means pixels).
