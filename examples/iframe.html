Embedding with an <iframe>

The simplest embed, available with zero JavaScript: point an <iframe> at a hosted simulation page. The sim runs fully isolated in its own document, so nothing collides with the host page.

<iframe src="https://your-host/bindingsims/ligand.htm"
        width="100%" height="900" title="Ligand binding simulation"></iframe>

Pick the page for the model you want: ligand.htm, homodimer.htm, ligands.htm, or receptors.htm. Append ?ext for the power-user features. Sizing is manual — set height to fit the sim.
